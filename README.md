# Builder Intelligence

An open-source AI-powered research toolkit for the residential homebuilding industry.

## Overview

Builder Intelligence is designed to help researchers, sales teams, marketers, and developers quickly gather and organize publicly available information about residential home builders, communities, floor plans, and related business data.

The project combines traditional data collection with AI-assisted analysis to reduce the time required to research builders and organize information for business use.

## Features

### Current Goals
- Builder research
- Community discovery
- Public contact collection
- Floor plan organization
- AI-assisted document analysis
- CSV and Excel export

### Planned Features
- Automated website crawling
- Community parser
- Floor plan PDF extraction
- Contact enrichment
- Salesforce export
- REST API
- CLI interface
- AI-powered research assistant

## Why This Project Exists

Residential builder information is spread across hundreds of company websites, PDFs, and listing services. Finding accurate information often requires significant manual effort.

Builder Intelligence aims to provide an open-source toolkit that automates much of this research while remaining flexible enough for developers and businesses to build upon.
